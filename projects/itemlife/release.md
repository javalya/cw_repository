## 应用商店信息

### App名称
**物品寿命 - 极简物品管理**

### 副标题
追踪每件物品的使用年限，做出更理性的消费决策

### 描述
```
物品寿命是一款专为极简主义者设计的物品管理工具。

核心功能：
• 记录物品购买日期和预期寿命
• 自动计算已使用时长和剩余天数
• 实时显示日均使用成本
• 到期前自动提醒更换
• 分类统计和数据分析

为什么选择物品寿命？
✓ 极简设计，3秒添加一件物品
✓ 本地存储，保护隐私
✓ 无广告，无社交，纯粹工具
✓ 支持数据导出，完全掌控

适合人群：
• 极简主义践行者
• 断舍离爱好者
• 理性消费者
• 租房族管理有限物品

开始更理性的消费生活，从记录第一件物品开始。
```

### 关键词
物品管理,物品寿命,极简主义,断舍离,消费决策,物品追踪,使用年限,理性消费
