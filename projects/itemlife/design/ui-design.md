## 组件规范

### 物品卡片 (ItemCard)
```
尺寸: 全宽 - 32px边距, 高度自适应
圆角: 12px
背景: #FFFFFF
阴影: 0 2px 8px rgba(0,0,0,0.05)
内边距: 16px

内容结构:
[图标 24x24] 名称 (Title)          [›]
             分类 (Caption)
             
寿命条: 高度6px, 圆角3px
       背景#E9ECEF, 进度色按状态

底部: 剩余时间 (Caption)    日均成本 (Caption)
```

### 主按钮 (PrimaryButton)
```
高度: 48px
圆角: 8px
背景: #2C3E50
文字: #FFFFFF, 16px Medium
状态:
  - 正常: 背景#2C3E50
  - 按下: 背景#1A252F
  - 禁用: 背景#6C757D
```

### 输入框 (InputField)
```
高度: 48px
圆角: 8px
边框: 1px #E9ECEF
背景: #FFFFFF
内边距: 12px 16px
文字: 16px Regular

状态:
  - 正常: 边框#E9ECEF
  - 聚焦: 边框#2C3E50
  - 错误: 边框#E74C3C
```
