# ItemLife - UI/UX 设计规范

**项目**: 物品寿命App  
**设计师**: UI/UX Designer  
**日期**: 2026-03-24  
**交付**: 3.27

---

## 设计原则

### 1. 极简主义
- 每屏不超过3个主要操作
- 留白充足，呼吸感强
- 减少装饰元素

### 2. 信息优先
- 核心数据（剩余天数、日均成本）突出显示
- 进度可视化（寿命条）
- 状态一目了然（颜色编码）

### 3. 快速操作
- 3秒添加一件物品
- 手势操作支持
- 减少输入步骤

## 色彩系统

### 主色调
```
Primary:    #2C3E50 (深蓝灰)     - 主按钮、标题
Secondary:  #34495E (灰蓝)       - 次级按钮
Background: #F8F9FA (浅灰白)     - 背景
Surface:    #FFFFFF (纯白)        - 卡片
```

### 状态色
```
健康:       #27AE60 (绿)         - 剩余>25%
警告:       #F1C40F (黄)         - 剩余<25%
紧急:       #F39C12 (橙)         - 剩余<7天
过期:       #E74C3C (红)         - 已过期
```

### 文字色
```
Primary:    #212529 (近黑)       - 主文字
Secondary:  #6C757D (灰)         - 次要文字
Tertiary:   #ADB5BD (浅灰)       - 辅助文字
```

## 组件规范

### 物品卡片 (ItemCard)
```dart
// 尺寸: 全宽 x 88px
// 圆角: 12px
// 背景: 白色
// 阴影: 0 2px 8px rgba(0,0,0,0.05)

结构:
┌─────────────────────────┐
│ [图标] 名称        [›]  │
│        分类 剩余时间     │
│ ████████░░░░ ¥12/天    │  ← 寿命条+成本
└─────────────────────────┘
```

### 寿命进度条 (LifespanBar)
```dart
// 高度: 6px
// 圆角: 3px
// 背景: #E9ECEF
// 进度色: 根据状态变色

状态色:
- >75%: #27AE60 (绿)
- 50-75%: #F1C40F (黄)
- 25-50%: #F39C12 (橙)
- <25%: #E74C3C (红)
```

### 分类标签 (CategoryChip)
```dart
// 高度: 32px
// 圆角: 16px (圆角胶囊)
// 内边距: 12px 16px
// 选中态: 填充主色
// 未选中: 边框样式
```
