## 设计规范概要

### 色彩系统
- 主色: #2C3E50 (深蓝灰)
- 状态色: 绿/黄/橙/红 (健康度指示)
- 背景: #F8F9FA (浅灰白)

### 字体规范
- 中文: PingFang SC
- 尺寸: Display(32) / Heading(24) / Title(18) / Body(16) / Caption(14) / Small(12)

### 页面结构
| 页面 | 核心布局 |
|------|----------|
| 首页 | AppBar + StatsCard + ExpiringSection + ItemList |
| 添加 | 表单布局（图片+名称+分类+日期+价格+寿命+备注） |
| 详情 | 图片 + 寿命圆环 + 成本卡片 + 详细信息 |
| 统计 | 概览卡片 + 饼图 + 折线图 + 即将到期列表 |
| 设置 | 分组列表（数据管理/提醒设置/外观/关于） |

### 组件规范
- ItemCard: 全宽卡片，88px高度，寿命条可视化
- LifespanBar: 6px高度，圆角，颜色随状态变化
- PrimaryButton: 48px高度，全宽，主色背景
- InputField: 48px高度，边框输入框
