# 财经早报内容审核清单

## 数据准确性检查 ✅

### 指数数据
- [ ] 道指、纳指、标普500涨跌幅是否准确
- [ ] 上证指数、深证成指、创业板指点位和涨跌幅
- [ ] 恒生指数、恒生科技指数涨跌幅
- [ ] 南向/北向资金流向

### 大宗商品
- [ ] 黄金价格（美元/盎司）
- [ ] 原油价格（布伦特/WTI，美元/桶）
- [ ] 涨跌幅方向（涨📈/跌📉）

### 宏观数据
- [ ] 美联储政策动向是否最新
- [ ] 汇率数据（美元指数、离岸人民币）
- [ ] 重要经济数据发布时间

## 内容质量检查 ✅

### 时效性
- [ ] 确保使用的是"过去24小时"的最新数据
- [ ] 周末/节假日需特别标注市场状态
- [ ] 避免使用过时的收盘数据

### 逻辑一致性
- [ ] 涨跌幅与点位变化方向一致
- [ ] 板块表现描述与市场整体走势吻合
- [ ] 宏观政策解读与数据匹配

### 格式规范
- [ ] 标题：财经早报 | YYYY年M月D日
- [ ] 所有数字使用阿拉伯数字
- [ ] 百分比保留1-2位小数
- [ ]  Emoji 使用恰当（📈📉）

## 常见问题 ⚠️

1. **数据滞后**：搜索结果可能包含前几天的新闻，需筛选最新数据
2. **休市误判**：港股/A股周末休市，美股周五数据是"隔夜"而非"今日"
3. **数字错误**：涨跌幅方向搞反（+写成-）
4. **来源混乱**：确保数据来自权威财经媒体
