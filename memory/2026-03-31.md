### 6. 集成 Superpowers 插件
**时间**: 15:45 - 15:55

刘总要求集成 Claude Code 的 Superpowers 插件，增强开发流程规范性：

#### Superpowers 简介：
- **定位**: Claude Code 插件，强制执行规范开发流程
- **核心价值**: 把"直接写代码"变成"先规划、再测试、后实现"

#### 核心技能：
| 技能 | 用途 |
|------|------|
| Brainstorm | 需求澄清，问关键问题 |
| Spec | 编写详细规格文档 |
| Plan | 任务分解（2-5分钟粒度） |
| TDD | 先写测试，再写实现（RED-GREEN-REFACTOR） |
| Debug | 系统性调试 |
| Review | 代码审查 |
| Git Worktree | 隔离开发分支 |

#### 安装方式：
```bash
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace
```

#### 使用判断：
- ✅ **启用**: 复杂功能（>3天）、需强制执行 TDD、系统性调试
- ❌ **不启用**: 简单功能（<1天）、紧急热修复

#### 触发关键词：
- `"help me plan this feature"` → Brainstorm + Plan
- `"let's debug this issue"` → Debug 技能
- `"implement this with TDD"` → TDD 流程
- `"review the code"` → Review 技能

- `skills/tech-lead/SKILL.md` - 新增 Superpowers 技能说明，更新技能调用策略和流程示例
