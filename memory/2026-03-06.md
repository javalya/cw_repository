# 2026-03-06 工作日志

## 网络问题排查与修复

### WARP 导致网络不稳定
- **问题**：Kimi Claw 连接故障，飞书消息无响应
- **根因**：Cloudflare WARP 模式劫持所有流量，导致 DNS 解析超时、延迟高达 360ms
- **解决**：改为 Proxy 模式 (`warp-cli mode proxy`)，默认不连接，需要时手动开启
- **结果**：网络延迟降至 6ms，服务恢复正常

### 服务器出口 IP 动态变化
- **发现**：服务器是字节跳动（火山引擎）云，出口 IP 池有多个 IP，每次请求可能走不同网关
- **影响**：微信公众号 IP 白名单频繁失效
- **记录IP**：101.126.54.134, 115.191.53.127, 115.191.56.155, 115.191.55.91, 115.191.48.190
- **解决**：将多个 IP 都添加到公众号白名单
